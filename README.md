# 🎬 Movie Recommender System

## Overview
This project is a Content-Based Movie Recommender System that suggests similar movies based on user input. It uses Machine Learning and NLP techniques to analyze movie metadata such as genres, keywords, cast, crew, and overview.

## Features
- Content-Based Movie Recommendation
- Cosine Similarity Algorithm
- Streamlit Web Application
- Fast and Accurate Recommendations
- User-Friendly Interface

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit
- Pickle

## How It Works
1. Data Preprocessing
2. Feature Engineering
3. Text Vectorization
4. Similarity Calculation using Cosine Similarity
5. Movie Recommendation Generation
